# 微信公众号发布手册

> 性质：凡涉及微信公众号发布的文章，均按此手册执行
> 更新方式：每次发布后发现新问题即补充
> 关联文件：`写作习惯与风格手册.md`（内容写作规范）、`AI时代战略者的产品思考框架_微信版.html`（参考样例）

## 一、完整发布流程

```
Step 1  撰写 Markdown 文档（按写作习惯与风格手册）
    ↓
Step 2  转换为 WeChat 兼容 HTML（按本手册 HTML 规范）
    ↓
Step 3  用 qwen-image-2.0-pro 生成配图（按本手册图像规范）
    ↓
Step 4  将图片以 base64 嵌入 HTML，替换 ASCII 图
    ↓
Step 5  添加底部模板（TA SHAN 署名 + logo + 二维码）
    ↓
Step 6  浏览器打开 HTML → 全选 → 复制 → 粘贴进微信编辑器
    ↓
Step 7  检查排版、填写标题摘要、发布
```

## 二、HTML 排版规范

```html

             font-size: 17px; line-height: 1.9; color: #333;
```

## 三、图像生成规范

### 3.1 使用的模型

| 模型 | API Key 来源 | 适用场景 |
|---|---|---|
| **qwen-image-2.0-pro**（首选） | 阿里云百炼：`YOUR_DASHSCOPE_API_KEY` | 中文文字图、信息图、结构图 |
| gpt-image-1（备用） | DMXAPI：`YOUR_DMXAPI_KEY` | 英文为主的极简图 |

**选择原则**：凡有中文文字的图，优先用 qwen-image-2.0-pro，中文字体渲染更准确。

---

### 3.2 API 调用格式（qwen-image-2.0-pro）

```python
import requests, base64

API_KEY = 'YOUR_DASHSCOPE_API_KEY'
ENDPOINT = 'https://dashscope.aliyuncs.com/api/v1/services/aigc/multimodal-generation/generation'

headers = {'Content-Type': 'application/json', 'Authorization': f'Bearer {API_KEY}'}

payload = {
    "model": "qwen-image-2.0-pro",
    "input": {
        "messages": [{"role": "user", "content": [{"text": "你的提示词"}]}]
    },
    "parameters": {
        "n": 1,
        "watermark": False,
        "prompt_extend": True,   # 开启智能提示词优化
        "size": "1024*1024"      # 可选：1024*1024、1024*1536 等
    }
}

resp = requests.post(ENDPOINT, headers=headers, json=payload, timeout=120)
img_url = resp.json()['output']['choices'][0]['message']['content'][0]['image']

# 下载保存
ir = requests.get(img_url, timeout=60)
with open('output.png', 'wb') as f:
    f.write(ir.content)
```

**注意**：图片 URL 有效期 24 小时，生成后立即下载保存。

### 3.4 提示词通用原则

- 开头固定：`简洁专业的信息图，适合微信文章，白色背景。`
- 明确标注文字内容（中文）
- 说明颜色方案（深蓝/浅蓝/白色/橙色）
- 结尾固定：`整体简洁商务风格。`
- qwen 会自动添加标题，无需在提示词里要求（`prompt_extend: True` 开启）

## 八.五、科教类文章引用规范

> 适用：科教类学术综述、认知科学、AI专题等引用外部文献的文章

### 格式标准：APA 第7版

**正文内引用：**
- 中文语境中直接命名作者时：`杜威（1933）`、`吉尔福德（1967）`
- 括号内引用时：`（Raichle et al., 2001）`
- 多作者（≥3人）：首次即用 et al. / 等人，如 `Gonçalves et al.（2025）`

**文末参考文献格式（APA 7th）：**

| 类型 | 格式 |
|---|---|
| 期刊论文 | 作者姓, 名首字母. (年). 文章标题. *期刊名*, *卷*(期), 起始页–结束页. https://doi.org/xxxxx |
| 专著 | 作者姓, 名首字母. (年). *书名*（版次，若适用）. 出版商. |
| 会议报告 | 作者. (年). 标题. 会议名称. [待核实格式] |

**完整性规则（强制）：**
1. 文内引用的所有文献，必须在文末列表中有对应条目
2. 文末列表中的所有条目，必须在文内有对应引用
3. 引用格式未确认（DOI 未核实）的，在条目末尾标注 `[待核实：具体卷期及DOI]`，不得完全省略该引用

**文末参考文献排列顺序：** 按第一作者姓氏字母顺序（A→Z），同一作者按年份升序
