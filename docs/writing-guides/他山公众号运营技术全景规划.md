## 二、当前可做的事（无需认证）

### 2.1 文章归档与分析

**目标**：把所有历史文章归档到本地，分析排版规范，为未来 AI 写作/发布提供样本。

**当前可行方案**：手动收集链接 → `links.txt` → `python import_links.py`

```bash
cd 项目群/feed-archiver
# 1. 把所有历史文章链接粘到 links.txt（每行一个）
# 2. 运行：
python import_links.py --source 他山学科交叉
```

文章保存在 `output/wechat/他山学科交叉/`，格式为带 YAML frontmatter 的 Markdown。

---

### 2.2 自定义菜单（可立即配置）

通过 API 程序化管理底部菜单，引导用户跳转：

```python
# 可配置的菜单项
菜单一：论坛（→ tashan.ac.cn/forum）
菜单二：活动报名（→ 当前活动页）
菜单三：关于我们（→ 官网介绍页）
```

**调用接口**：`POST /cgi-bin/menu/create`

**价值**：不需要人工进后台改菜单，可以在代码里随时更新，配合活动节奏。

---

### 2.3 带参二维码生成（渠道追踪）

为不同渠道生成独立二维码，精确追踪用户关注来源：

| 场景 | 二维码 | 用途 |
|---|---|---|
| 线下活动海报 | `scene=activity_2026_q1` | 活动现场扫码关注 |
| 文章内嵌 | `scene=article_001` | 文章内关注来源 |
| 合作伙伴推荐 | `scene=partner_xxx` | 来自谁的推荐 |

**调用接口**：`POST /cgi-bin/qrcode/create`（永久二维码）

---

### 2.4 消息自动回复（需配置服务器）

用户给公众号发消息时，触发自动回复。服务器已有（`101.200.234.115`）。

| 触发词 | 自动回复内容 |

| `报名` | 返回当前活动报名链接 |

| 关注事件 | 发送欢迎语 + 推荐文章 |

**配置方法**：公众号后台 → 消息推送 → 配置服务器 URL + Token

## 三、认证后可解锁（建议推进）

**认证方式**：以"中国科学院大学他山学科交叉创新协会"名义提交组织证明认证。

**认证成本**：300元/年（微信官方认证费）

### 3.1 群发接口（最高价值）

- 每天可群发 1 次给所有关注者
- 可程序化发布：从 Cursor 写完文章 → 自动上传素材 → 自动群发，全程无人工操作
- 调用接口：`POST /cgi-bin/message/mass/sendall`

**实现路径**：
```
AI 生成文章 → 上传图文素材 → 调群发接口 → 发布完成
```

### 3.2 数据统计（运营决策支撑）

每篇文章自动采集：

| 指标 | 接口 |
|---|---|
| 阅读量 | `GET /datacube/getarticlesummary` |
| 点赞数 | 同上 |
| 分享次数 | 同上 |
| 取关人数 | `GET /datacube/getuserloss` |
| 新关注人数 | `GET /datacube/getusersummary` |

**价值**：每篇文章发出后自动生成数据报告，不用人工去后台查。

### 3.3 用户管理（粉丝分析）

- 获取全量粉丝 openid 列表
- 按标签分组（如：实训营学员 / 一般关注者 / 合作伙伴）
- 为特定分组发送定向消息

### 3.4 客服消息（24小时窗口回复）

用户发消息后 48 小时内可以主动推消息给他（不走群发限制）。
适合：报名确认、活动提醒、个性化回复。

### 3.5 草稿 + 发布 API（自动化发布流）

```
写文章（AI 辅助）
    → POST /cgi-bin/draft/add（新建草稿）
    → POST /cgi-bin/freepublish/submit（提交发布）
    → GET /cgi-bin/freepublish/get（查询发布状态）
```
